Day 32 - 40 Days of JavaScript - Modules

The js modules is the concept of a way to code where code base can be divided into smaller chunk rather than a gigantic code base file.There are several types of module export. There are:

    
    
      
  • Named export/import:
        export { add, sub };
        import { add, sub } from "./calculate.js";
      
    • 
      
  • Named export/import with alias:
        export { add, sub };
        import { add as plus, sub as minus } from "./calculate.js";
      
    • 
      
  • Default export/import: (type 1)
        export default sayHello;
        import {default as anyName} from "./sayHello.js";
      
    • 
      
  • Default export/import: (type 2)
        export default sayHello;
        import anyName from "./sayHello.js";
      
    • 
      
  • Namespace export/import: 
        export {add, sub};
        import * as calc from "./calculate.js";
      
    • 
      
  • Combined export/import:
        All the modules export from a single files is called Combined module export. Here a separate single file combined.js is created and export modules of calculate.js and sayHello.js form combined.js. 

        in combined.js
        export * as calc from "./calculate.js";
        export {default as hello} from "./sayHello.js";

        in index.js 
        import * as combine from "./combined.js";
      
    • 
      
  • Dynamic import:
        The dynamic import is the way of import multiple modules using Promise APIs.
        // Dynamic import
        const { add , sub } = await import( "./calculate.js");


        // Dynamic import of multiple modules
        const promise = Promise.all(
          [
            await import("./calculate.js"),
            await import("./greetings.js")
          ]
        );

        promise.then(result => {
          const calc = result[0];
          const greet = result[1];

          console.log(calc.add(5, 5));
          console.log(calc.sub(10, 5));
          
          greet.sayHi();
          greet.sayHola();
        });
